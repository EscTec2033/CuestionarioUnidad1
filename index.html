Cuestionario Sobre el contenido de la Unidad I.

  1. ¿Qué es un Sistema Operativo?

    2. Un sistema operativo es un programa informático que administra los recursos del equipo y facilita la interacción con el usuario. En otras palabras,
    es el software fundamental que coordina y dirige todos los servicios y aplicaciones que utilizas la computadora.

  2. Componentes de un Sistema Operativo
    3. Núcleo (Kernel):
    El núcleo es el corazón del sistema operativo. Administra recursos como la memoria, el procesador, los dispositivos de entrada/salida
    y el sistema de archivos
    Sistema de Archivos:
    Organiza y gestiona los archivos en el almacenamiento.
    Interpretación de comandos.
    Tiene como función comunicar las órdenes dadas por el usuario en un lenguaje que el hardware pueda interpretar (sin que aquel que dé las órdenes conozca dicho lenguaje).
  3. Tareas del Sistema Operativo

    4. Gestionar las entradas y salidas de datos a través de los periféricos: Además de direccionar las entradas y salidas de datos, el sistema operativo proporciona controladores (drivers) para interactuar con los periféricos de entrada y salida, como teclados, mouse, impresoras, discos duros externos, entre otros.

  4. Funciones de un Sistema Operativo

    • Gestionar la memoria de acceso aleatorio y ejecutar las aplicaciones,
      designando los recursos necesarios.

    • Administrar al CPU gracias a un algoritmo de programación.

    • Direccionar las entradas y salidas de datos (a través de drivers) por
      medio de los periféricos de entrada o salida.

    • Administrar la información para el buen funcionamiento de la PC.

    • Dirigir las autorizaciones de uso para los usuarios.

    • Administrar los archivos.

  5. Características de un Sistema Operativo.

    • Es el intermediario entre el usuario y el hardware.

    • Es necesario para el funcionamiento de todos los computadores, tabletas y teléfonos móviles.

    • Otorga seguridad y protege a los programas y archivos del ordenador.

    • Está diseñado para ser amigable con el usuario y fácil de usar.

    • Permite administrar de manera eficiente los recursos del ordenador.

    • La mayoría requiere del pago de una licencia para su uso.

    • Permite interactuar con varios dispositivos.

    • Es progresivo, ya que existen constantemente nuevas versiones que se
      actualizan y adaptan a las necesidades del usuario.

  6. Tipos de Sistema Operativo.

    7. Los tipos de sistema operativo varían según el hardware y la función de cada dispositivo. Existen algunos para ordenadores y otros para dispositivos móviles:


    • Según el usuario pueden ser: multiusuario, sistema operativo que permite que varios usuarios ejecuten simultáneamente sus programas; o monousuario,
      sistema operativo que solamente permite ejecutar los programas de un usuario a la vez.

    • Según la gestión de tareas pueden ser: monotarea, sistema operativo que solamente permite ejecutar un proceso a la vez; o multitarea, sistema operativo
      que puede ejecutar varios procesos al mismo tiempo.

    • Según la gestión de recursos pueden ser: centralizado, sistema operativo que solo permite utilizar los recursos de un solo ordenador; o distribuido,
      sistema operativo que permite ejecutar los procesos de más de un ordenador al mismo tiempo.

  7. Sistemas Operativos Para PC.

    8. Un sistema operativo (SO) es el programa que, después de ser cargado inicialmente en la computadora por un programa de arranque, administra todos los demás programas de aplicación en una computadora. Los programas de aplicación hacen uso del sistema operativo al realizar solicitudes de servicios a través de una interfaz de programa de aplicación (API) definida. Además, los usuarios pueden interactuar directamente con el sistema operativo a través de una interfaz de usuario, como una interfaz de línea de comandos (CLI) o una interfaz de usuario gráfica (GUI).

  8. Sistemas Operativos Móviles

    9. Un sistema operativo móvil o (SO) móvil, es un sistema que controla un dispositivo móvil al igual que los PCs que utilizan Windows o Linux, los dispositivos móviles tienen sus sistemas operativos como Android, IOS entre otros. Los sistemas operativos móviles son mucho más simples y están más orientados a la conectividad inalámbrica. A medida que los teléfonos móviles crecen en popularidad, los sistemas operativos con los que funcionan adquieren mayor importancia.

  9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

    10. Los principales tipos de sistemas operativos se dividen en monotarea y multitarea. Los sistemas operativos modernos son del segundo tipo, es decir, son capaces de realizar varias tareas al mismo tiempo. Esto también incluye a los 3 sistemas operativos más populares: Microsoft Windows, Linux, MacOS.

  10. Compatibilidad de los Sistema Operativo

    11. La compatibilidad de los sistemas operativos se refiere a la capacidad de un dispositivo o programa para funcionar correctamente con un determinado sistema operativo. Esto implica que los componentes y software de un dispositivo deben ser capaces de interactuar y comunicarse de manera eficiente con el sistema operativo instalado.

  11. ¿Cuál es la función básica de un Sistema Operativo?

    12. Un sistema operativo es el software principal que permite la interacción entre el usuario y la computadora. Sus funciones principales son:

    • Gestión de procesos: Administra los recursos necesarios para que los programas se ejecuten correctamente, como la memoria y el tiempo del CPU.

    • Gestión de la memoria: Controla cómo se asigna y libera la memoria RAM para los procesos.

    • Comunicación entre aplicaciones: Facilita la interacción entre programas y procesos.

    • Gestión de archivos: Administra el acceso a archivos y directorios.

    • Interfaz con el usuario: Proporciona una forma visual o de línea de comandos para que el usuario interactúe con la computadora.

  13. ¿Qué es un sistema operativo multitareas?

    14. Es aquel que puede ejecutar múltiples procesos al mismo tiempo, permitiendo que los usuarios realicen varias tareas simultáneamente en sus dispositivos1. Estos sistemas son esenciales para aumentar la eficiencia y la productividad tanto en el trabajo como en el uso personal de los dispositivos electrónicos. En términos más técnicos, los sistemas operativos multitarea permiten que varios programas o aplicaciones se ejecuten aparentemente al mismo tiempo, compartiendo uno o más procesadores.

  14. ¿Qué es un sistema operativo multiusuario?

    15. Es aquel que permite que múltiples usuarios accedan a un solo sistema con un SO. En otras palabras, varios usuarios pueden interactuar con la misma computadora o terminal de forma pseudo-simultánea, compartiendo recursos y procesamiento1. Si tienes más preguntas, no dudes en hacerlas.

  15. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

    16. Es aquel que utiliza una interfaz visual para que los usuarios puedan interactuar con el sistema de manera más intuitiva y visual. A diferencia de los sistemas operativos basados en línea de comandos, que requieren el uso de comandos de texto, los sistemas operativos de ambiente gráfico permiten realizar acciones a través de iconos, ventanas y menús.

  16. ¿Cuándo inicia el funcionamiento del Sistema Operativo?

    17. Una vez que se ha colocado el sistema operativo en la memoria principal, el cargador de arranque ordena al procesador que ejecute una instrucción de salto a dicha área de memoria. En ese momento, el sistema operativo asume el control y comienza a gestionar las actividades de la máquina1. Si tienes más preguntas o necesitas más detalles, no dudes en preguntar.

  17. Mencione las dos interfaces que forman parte del Sistema Operativo

    18. Las dos interfaces de usuario que forman parte de un sistema operativo son la Interfaz de Línea de Comandos (CLI) y la Interfaz Gráfica de Usuario (GUI)1. La CLI permite a los usuarios interactuar con el sistema operativo mediante comandos de texto, mientras que la GUI proporciona una interfaz visual con ventanas, iconos y elementos gráficos para facilitar la interacción.

  18. ¿Qué es un proceso en entorno de los sistemas operativos?

    19. Un proceso en el ámbito de los sistemas operativos se refiere a un programa en ejecución. Es una entidad activa que realiza tareas específicas dentro del sistema operativo1. Cada proceso tiene su propio espacio de memoria, recursos asignados y estado de ejecución. Básicamente, los procesos son todos los programas o tareas que están ejecutándose en el sistema operativo.

  19. ¿Qué es un sistema operativo distribuido y centralizado?

    20. Un sistema operativo distribuido es un conjunto de programas informáticos que utilizan recursos computacionales en varios nodos de cálculo distintos para lograr un objetivo compartido común. Estos sistemas, también conocidos como “computación distribuida” o “bases de datos distribuidas”, utilizan nodos diferentes para comunicarse y sincronizarse a través de una red común. Los nodos pueden representar dispositivos de hardware físicos o procesos de software diferentes. En un sistema distribuido, la responsabilidad se comparte entre múltiples nodos, lo que permite una mayor escalabilidad y redundancia. Por otro lado, un sistema operativo centralizado tiene una estructura de un solo nivel, donde todos los componentes dependen de un único elemento de control. En este caso, todos los cálculos los realiza un solo ordenador en una ubicación. La principal diferencia entre un sistema centralizado y uno distribuido radica en el patrón de comunicación entre los nodos del sistema. En un sistema centralizado, el estado se encuentra dentro de un nodo central al que los clientes acceden a través de un método personalizado2. Sin embargo, este enfoque puede tener un punto de error único y puede sobrecargar la red si todos los nodos acceden al nodo central.

  20. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

    21. Los sistemas de archivos son estructuras utilizadas por los sistemas operativos para organizar y almacenar datos en dispositivos de almacenamiento, como discos duros o unidades USB. Permiten a los usuarios guardar, acceder y gestionar sus archivos de manera eficiente1. A continuación, te presento algunos de los sistemas de archivos más importantes utilizados por diferentes sistemas operativos:


    • FAT32 (File Allocation Table 32): Establecido en 1996, es uno de los sistemas más antiguos. Aunque tiene limitaciones en el tamaño máximo de archivo y partición, sigue siendo compatible con muchos dispositivos y sistemas operativos.

    • exFAT (Extended File Allocation Table): Diseñado para dispositivos de almacenamiento extraíbles como memorias USB y tarjetas SD. Ofrece una mayor compatibilidad con archivos grandes y no tiene las restricciones de FAT32.

    • NTFS (New Technology File System): Utilizado principalmente en sistemas Windows. Proporciona características avanzadas como permisos de acceso, compresión y cifrado de archivos.

    • HFS+ (Hierarchical File System Plus): Antiguo sistema de archivos de macOS. Ha sido reemplazado por APFS (Apple File System) en versiones más recientes.

    • APFS (Apple File System): El sistema de archivos predeterminado en macOS desde macOS High Sierra. Optimizado para unidades SSD y ofrece características como snapshots y clonación de archivos.

    • ext2, ext3 y ext4: Utilizados en sistemas Linux. Ext2 es el más antiguo, ext3 añade registro de diario para mayor seguridad, y ext4 es la versión más moderna con mejor rendimiento y escalabilidad.


  21. Mencione otras de las características actuales de los sistemas operativos

    • Gestión de procesos: Los sistemas operativos planifican y administran las tareas para que sean manejadas por la computadora.

    • Administración de memoria: El sistema operativo gestiona la memoria RAM, asignando y liberando espacio para los programas en ejecución.

    • Control de dispositivos de entrada y salida: Permite la comunicación entre la computadora y periféricos como teclados, ratones, impresoras y discos duros.

    • Interfaz gráfica de usuario (GUI): Proporciona una forma visual e intuitiva para que los usuarios interactúen con la computadora mediante ventanas, iconos y menús.

    • Protección de datos: Los sistemas operativos implementan medidas de seguridad para proteger archivos y datos contra accesos no autorizados.


  22. ¿Que permite la exigencia de muchos controladores en un sistema operativo?

    23. Los controladores (también conocidos como drivers) son componentes de software que permiten que el sistema operativo y los dispositivos se comuniquen entre sí. Cuando una aplicación necesita leer datos de un dispositivo, llama a una función implementada por el sistema operativo. Luego, el sistema operativo invoca una función implementada por el controlador específico para ese dispositivo. El controlador, generalmente desarrollado por el fabricante del dispositivo, sabe cómo comunicarse con el hardware del dispositivo para obtener los datos. Una vez que el controlador obtiene los datos, los devuelve al sistema operativo, que luego los transmite a la aplicación. En resumen, los controladores permiten que el sistema operativo gestione una amplia variedad de dispositivos de hardware, como impresoras, tarjetas de red, tarjetas gráficas, etc.
